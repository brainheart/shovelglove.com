The video file format is quicktime. I'm not sure how
well/quickly that will play for most people, but I
didn't know what else to do. If there is a better
format I ought to be using so that more people can use
it, let me know. They are small 10-20 second clips,
one per movement.

For people who can't get the quicktime to work, I've
exported each movie as a sequence of stills, at 3
stills per second. You can navigate the stills (and
see the quicktime movies) from the links below. Note
that this is extremely boring, but hopefully it should
give you some idea of the movements if you are
completely lost. The stills need some editing. Every
once in a while you'll see a completely blacked out
frame. Not sure why, hadn't had time to look into it
yet.
