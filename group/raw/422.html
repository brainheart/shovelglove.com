Hi Michael, Deb,

The only hard rule about timing is do it every N day.
That being said, I think it's easier if you have a set
time (mine is morning) because you're less likely to
weasel out. It doesn't have to be set in stone, but
just a usual, default time. Morning, for me has the
following advantages:

1) Post workout I feel energized and like I've
accomplished something. Going to work is less
depressing.

2) If I do miss it, there's always evening.

The problem with making evening your set time is that
it's too easy to convince yourself that it's more
important to just go to sleep.

Reinhard
